Archive Structural Platform

데이터는 구조로 연결되고 흐름으로 작동하며 시간을 축적한다.

—

1. 데이터 구조
   작품 / 텍스트 / 이미지 / 보고서 분리 저장
   ID 기반 구조화
   standard / zoom 이중 이미지 구조

2. 연결 방식
   작가 → 작품 → 전시 → 아카이브
   JSON 기반 연결 구조
   단일 페이지 다층 연결

3. 저장 방식
   코드 / 이미지 저장소 분리
   정적 데이터 + 캐시 구조
   버전 기반 시간 축적

4. 사용자 흐름
   ENTRY → 구조 선택 → 데이터 진입
   이미지 → 전체보기 → 아카이브 연결
   텍스트 → 보고서 → 구조 이해

5. 확장 구조
   작가 / 갤러리 / 기관 통합
   Archive Pro 연동
   플랫폼 → 시간 기반 시스템 확장

DCAP-01

Archive Structural Platform
아카이브 구조 플랫폼

Data connects as structure, operates as flow, and accumulates time.

—

1. Data Structure
   Separate storage: works / text / images / reports
   ID-based structure
   Dual image system (standard / zoom)

2. Connection
   Artist → work → exhibition → archive
   JSON-based linkage
   Multi-layer single-page connection

3. Storage
   Separate code and image repositories
   Static data + cache
   Version-based time accumulation

4. User Flow
   ENTRY → structure → data access
   Image → view → archive link
   Text → report → structural understanding

5. Expansion
   Integration of artists / galleries / institutions
   Archive Pro linkage
   Platform evolving into time-based system
